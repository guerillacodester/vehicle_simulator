# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-10-24

### Added

- Initial release of strapi-plugin-action-buttons
- Custom action button field type for Strapi v5
- Configurable button labels via schema options
- Custom handler functions via window object
- JSON metadata storage for tracking button actions
- Visual feedback for execution state
- Display of last action metadata
- TypeScript support
- Comprehensive documentation with examples
- Multiple real-world use case examples

### Features

- Dynamic button labels (supports emojis and unicode)
- Per-instance handler configuration
- Async handler support
- Error handling with user feedback
- Integration with Strapi Design System
- Works with any content type (API or plugin)
- Zero external dependencies

### Documentation

- Complete README with installation instructions
- Quick start guide
- Usage examples (email, CSV upload, CRM sync)
- Advanced patterns (handler factories, progress indicators)
- Best practices and troubleshooting guide
- API reference
